| This document, based on April Dunford\'s \"Positioning Show,\" clarifies the critical distinction between differentiated value and objection handling in B2B technology sales and marketing. It emphasizes that true value drives purchasing decisions, while objection handling removes barriers to a sale. The document outlines how to identify each, how they function differently in marketing and sales strategies, and how they relate to various personas within a B2B buying committee. It concludes by acknowledging exceptions where objection handling can become a core differentiator, particularly when an incumbent vendor has significant shortcomings in a specific area. |

| **How Differentiated Value is Defined**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Competitive Alternatives:** Positioning begins by identifying competitive alternatives, answering the question: \"What would a customer do if you didn\'t exist?\"                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|   - **Status Quo:** This includes what the customer is currently using (e.g., Excel, manual processes, old legacy systems, or a previous product that failed).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|   - **Shortlist Competitors:** These are other products or solutions that would be considered alongside yours.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Identifying Differences:** After identifying alternatives, the next step is to determine what makes your offering different -- what you have that alternatives don\'t. This includes features and capabilities of both the product and the company (e.g., pricing models, professional services, adherence to standards like SOC 2 or HIPAA).                                                                                                                                                                                                                                                                                                                                           |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Translating Capabilities to Value:** Not all differentiated capabilities are equally important. The \"so what?\" test is applied to features: \"Why does a customer care? What is the value that this feature enables for a customer\'s business?\" The goal is to distill a long list of differentiated capabilities into one, two, or three core value themes or buckets that customers can easily remember.                                                                                                                                                                                                                                                                          |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ### Identifying Objection Handling vs. Pure Value                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Beyond Value Themes:** Some important aspects don\'t fit neatly into core value themes but are crucial for winning deals, often because customers raise them in sales calls.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Customer Support Example:** Excellent customer support is vital for renewals but may not be \"pure value\" for a customer initially looking to buy. If great support is the *only* differentiator, it\'s unlikely to drive a purchase on its own. It\'s also hard to objectively measure \"better\" support unless there are specific, measurable advantages (e.g., 24/7 support for international companies).                                                                                                                                                                                                                                                                          |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Ease of Adoption Example:** Investments in Professional Services, pre-built integrations, or special training to make software adoption easy often fall into objection handling rather than pure value.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ### What is Objection Handling?                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Addressing Concerns:** Objection handling addresses customer concerns that could prevent a sale, even if the customer likes the product\'s core offering. Examples include:                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|   - Lack of adherence to a specific standard.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|   - Perceived difficulty of deployment or use.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|   - Pricing model or affordability concerns.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|   - Concerns about support for diverse users or time zones.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Not the Buying Reason:** These are not typically the reasons a customer *buys* a product, but they are reasons a customer might *not buy* it. While important and potentially differentiated, they are usually not sufficient to convince a customer to purchase on their own.                                                                                                                                                                                                                                                                                                                                                                                                          |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ### How to Position Value vs. Objections in Marketing and Sales                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Marketing:**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|   - **Core Value Front and Center:** Differentiated value should be prominently displayed on the website\'s homepage (e.g., accurate revenue predictability for CRM software).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|   - **Objection Handling Deeper:** Objection handling points (e.g., special training for ease of use) should be placed deeper within the website, mentioned only after the core value has been established, to avoid raising questions about the product\'s inherent difficulty.                                                                                                                                                                                                                                                                                                                                                                                                            |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Sales:**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|   - **Sales Pitch Focus:** The sales pitch should primarily revolve around the differentiated value that no one else can deliver.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|   - **Unspoken Objection Handling:** A dedicated step for handling unspoken objections should be included near the end of the sales pitch. This allows the sales team to proactively address potential concerns (e.g., ease of adoption for sales reps) before the customer leaves with unresolved doubts.                                                                                                                                                                                                                                                                                                                                                                                  |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ### Understanding the Roles in a B2B Buying Committee                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Complex Buying Process:** B2B purchases, especially for enterprise software, involve a \"deal team\" of typically five to nine people, not a single decision-maker.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Champion:** This persona (e.g., Head of Sales) is responsible for creating a shortlist, evaluating options, and recommending a solution to the economic buyer. The deal cannot progress without the champion\'s buy-in.                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **\"Department of No\":** Other personas (e.g., Legal, IT, end-users, purchasing) can kill a deal by raising objections, even if they don\'t drive the purchase themselves.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ### Mapping Value to Champions and Objections to \"The Department of No\"                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Value for Champion/Economic Buyer:** Differentiated value must strongly resonate with the champion and economic buyer to get on the shortlist and drive the deal forward.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Arming the Champion:** It\'s crucial to equip the champion with information and responses to handle the potential objections from other members of the buying committee (e.g., IT concerns about SOC 2 compliance, legal concerns).                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Targeted Content:** Specific content can be created for these \"department of no\" personas, but it should focus on addressing their objections rather than promoting core value.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Deal Drivers vs. Deal Killers:** Differentiated value gets you the deal; objection handling prevents you from losing it.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ### The Exception: When Objection Handling Becomes the Differentiated Value                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Incumbent Vendor Failure:** In rare cases, an incumbent vendor\'s significant and widespread failure in a specific area (e.g., poor customer support, buggy software) can elevate what would typically be an objection handler into a core differentiated value.                                                                                                                                                                                                                                                                                                                                                                                                                        |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **ERP Example:** An ERP vendor for midsize manufacturing companies capitalized on an incumbent\'s poor customer support and buggy platform. Their \"rock-solid support\" became a central value proposition, backed by specific guarantees (dedicated support person, 2-hour SLA), third-party code quality reports, and high customer satisfaction testimonials.                                                                                                                                                                                                                                                                                                                         |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| - **Strategic Focus:** This exception allows a competitor to focus on a segment of the market where the incumbent is failing, using that specific area as a primary value driver.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
